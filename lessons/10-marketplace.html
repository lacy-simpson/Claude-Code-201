Find skills others have built

The UiPath Enterprise AI & Automation Marketplace

You do not have to build every skill yourself. The UiPath Enterprise AI & Automation Marketplace is a central place to discover skills, agents, and prompt libraries that teams across the company have built and vetted. This is a quick tour so you know where to look.

Lesson Objective

Describe what the marketplace is, find a skill in it, and install one using any of the three methods.

What it is

Discover and Deploy Reusable Assets

The marketplace is a centralized catalog of reusable AI assets: skills, automation agents, prompt libraries, and frameworks, vetted by your AI Automation Office. You browse by business domain, see what was published recently, or search for something specific.

A skill's page

What You See on a Skill

Each skill has a detail page so you can decide whether it fits before you install. Expect to find:

Overview

A plain description, business use cases, supported platforms, and tags so you can tell at a glance what it does and who it is for.

Versions & Docs

The current version, version history, and documentation, so you know what you are installing and how to use it.

Reviews

What other people thought, plus the owner and when it was last updated.

Getting a skill

Three Ways to Install

A skill's page offers a few install methods. Pick the one that matches your setup.

  • Download ZIPNo repository access needed. Download the ZIP, unzip it, and install from the folder. This is the method from the last lesson.
  • Full CatalogDownload every skill in one bundle. Handy when you are getting started and want the whole set at once.
  • MarketplaceIf you have access to the internal repository, add the marketplace once, then install bundles by name. Updates flow through automatically.
Download ZIP, then install 
/plugin install ~/Downloads/<skill-folder>
Marketplace: add once, then install bundles 
/plugin marketplace add <marketplace>
/plugin install <bundle>@<marketplace>
Same habits apply

A skill from the marketplace is still code that acts on your behalf. Read its description, start with read-only skills, and review what it does the first time you run it, exactly as you would for a skill you built.
