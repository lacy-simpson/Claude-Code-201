Put a skill to work

Running a Skill

A skill is only useful once you run it. The good news is that running one is easy: install it, then ask for it by name. Let's get a skill onto your machine and watch Claude work through it.

Lesson Objective

Install a skill the simplest way, run it, and read what Claude is doing while it runs.

Get a skill

Install One the Easy Way

The simplest path needs no extra setup: download a skill as a ZIP, unzip it, and install it with one command. Start with a read-only skill so the first run cannot change anything.

  • Download and unzipDownload the skill's ZIP and extract it, for example into your Downloads folder.
  • Install itIn Claude Code, run the install command pointing at the unzipped folder.
Install a downloaded skill 
/plugin install ~/Downloads/meeting-notes-summarizer

Lesson 10 tours the marketplace where these skills come from, with more ways to install.

Run it

Ask for It by Name

Once a skill is installed, you run it by asking. Claude recognizes the request and follows the skill's playbook.

Run a skill 
Run the meeting-notes-summarizer skill on the notes in this folder.

You do not have to remember exact steps. The skill carries them, so a short request is enough.

Watch it work

Following Along

A skill can take a moment, especially if it reads several files. You do not have to sit and watch. A few things in the interface help you keep track.

  • Expand the stepsClick into the messages to see what Claude is doing at each step, such as which files it is reading.
  • Let it run in the backgroundSome skills launch parallel helpers to work through many items at once. You can start another session and come back.
  • Watch for the attention dotA session that needs you shows a dot in the session list. Open it, see what it needs, and continue.
  • Review and closeWhen it finishes, read the output, then mark the session complete or move on.
Key Takeaway

Install once, then run a skill with a plain-language request. Start with read-only skills, follow along by expanding the steps, and review the result before you use it.
