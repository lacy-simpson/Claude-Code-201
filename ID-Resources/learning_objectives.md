# Learning Objectives — Claude 101: Orient, Access, and Activate

Course: Claude 101: Orient, Access, and Activate
Audience: All UiPath employees
Format: Facilitator-led webinar, 50 min
Status: DRAFT — review against RESEARCH_SUMMARY.txt once complete

---

## Session Goal

Participants leave knowing what Claude is, how to get access, how to set up Claude with basic work context, how to use one simple chat loop, and what to learn next. This session is orientation and activation — deeper skill-building comes in later sessions.

---

## Learning Objectives

By the end of this session, participants will be able to:

### Bloom's Level: Remember / Understand

1. **Describe** what Claude is and how it differs from a search engine or a simple chatbot.
   *Assessed by: verbal Q&A, follow-along activity response*

2. **Identify** the three Claude tools available at UiPath (Chat, Cowork, Code) and state which audience each is designed for.
   *Assessed by: facilitator check-in question during Tools segment*

3. **Recall** how to request access to Claude at UiPath and who to contact for access issues.
   *Assessed by: leave-behind reference review*

### Bloom's Level: Apply

4. **Use** Claude Chat for a low-risk first task by giving a goal, context, desired output, and one review cue.
   *Assessed by: follow-along activity — start a simple chat and send one follow-up*

5. **Configure** at least one durable work context setting in Claude Chat, such as Instructions for Claude or a new Project.
   *Assessed by: facilitator prompt during setup segment*

### Bloom's Level: Understand (Transfer)

6. **Explain** the difference between a regular chat and a Project, and describe one scenario from their own work where a Project would be more useful.
   *Assessed by: facilitator debrief question after Demo segment*

---

## Alignment Notes

- Objectives 1-3: Covered in Opening, Claude Introduction, Access, and Tools segments
- Objective 4: Covered in Use Claude Chat for a First Task segment
- Objective 5: Covered in Set Up Claude for Your Work segment
- Objective 6: Reinforced in Demo segment — the Project context payoff

All objectives are achievable within the 50-minute session. Objectives 4 and 5 require participants to have access to Claude during the session — confirm access provisioning before the live date.

---

- Claude Cowork and Code hands-on use
